#  Business Sales Performance Dashboard

##  Project Overview
This Power BI dashboard analyzes business sales performance using interactive visualizations and KPIs.

## 📊 KPIs
- Total Revenue
- Total Orders
- Total Customers
- Total Quantity

##  Dashboard Features
- Revenue by Country
- Revenue by Month
- Revenue by Product Description
- Quantity Analysis
- Customer-wise Revenue
- Interactive Filters (Year, Country, Month)
